# Vue 3 + Webpack 5 项目

## 环境要求
- webpack@5.78.0
- vue@3.4.34

## 安装依赖
```bash
npm install
```

## 开发模式
```bash
npm run dev
```

## 生产构建
```bash
npm run build
```

## 项目结构
```
npm-webpack-test/
├── public/
│   └── index.html          # HTML 模板
├── src/
│   ├── App.vue             # 根组件
│   └── main.js             # 入口文件
├── package.json            # 项目配置
├── webpack.config.js       # Webpack 配置
└── README.md              # 说明文档
```
